# Phishing Email Detection & Awareness System

## Objective
This repository contains a professional Security Advisory and Phishing Awareness Report created as part of a cybersecurity internship. The goal of this task was to analyze a malicious email, identify indicators of compromise (IoCs), and create a non-technical awareness document to educate employees on how to spot and prevent phishing attacks.

## Analysis Methodology
*   **Indicator Identification:** Analyzed the email body for psychological triggers (urgency/fear), generic greetings, vague sign-offs, and deceptive destination URLs.
*   **Header Analysis:** Conceptually analyzed email headers to identify Sender Policy Framework (SPF) and DomainKeys Identified Mail (DKIM) failures, definitively proving the email was spoofed.
*   **Awareness Drafting:** Translated technical findings into actionable, business-friendly guidelines (Do's and Don'ts) for general staff.

## Deliverables
*   **[Phishing_Awareness_Report.pdf](./Email-analysis-report.pdf)** - The final security advisory document detailing the anatomy of the attack, technical evidence, and employee prevention guidelines.
